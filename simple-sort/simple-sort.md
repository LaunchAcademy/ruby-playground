# Simple Sort

## Instructions

Write a ruby program which sorts numbers. You are not allowed to use any
of the native sort methods found in ruby, you have to write your own!

INPUT SAMPLE:

```ruby
"30.02 -88.87 10.58 -99.22 107.33"
```

OUTPUT SAMPLE:

```ruby
"-99.22 -88.87 10.58 30.02 107.33"
```
